2. **Verify translations**: We use translations also in public domain
3. **Cross-reference**: All sources are cited and verifiable

### Spurgeon Verification
- Original work published 1866 (159 years ago)
- USA public domain: works published before 1929

## üìù File Format

All JSON files follow this structure:

```json
{
  "metadata": {
    "title": "Work Title",
    "author": "Author Name",
    "published": "1866",
    "status": "Public Domain",
    "source": "Original source URL",
    "translation": "Translator name (if applicable)"
  },
  "entries": [
    {
      "date": "01-01",
      "title": "Reading Title",
      "scripture": "Reference (if applicable)",
      "text": "Full reading text..."
    }
  ]
}
```
